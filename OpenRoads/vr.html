Play SkyRoads in Virtual Reality

OpenRoads supports the Oculus Rift DK2 and DK1 using NodeJS. Download the premade bundle for Windows (32-bit and 64-bit) to try it out. Linux and OSX support are planned. You can always try building and installing yourself, pull requests are welcome.


Click Here to Download SkyRoads VR

How to Play

  1. Download SkyRoads VR
  2. Extract the ZIP Archive
  3. Install the Visual C++ Redistributable Packages for Visual Studio 2013 if you do not have them already.
  4. Inside your freshly extracted SkyRoadsVR folder, run either Skyroads VR or Skyroads XMas VR
  5. Enjoy!
